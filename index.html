HTML5プロフェッショナル
認定試験とは

HTML5,CSS3,JavaScriptなど最新のマークアップに関する技術力と知識を中立的な立場で認定する認定資格

試験で問われるスキルと知識

この資格がおすすめの職種

  • マークアップエンジニア
  • システムエンジニア
  • デザイナー
  • Webディレクター

マークアップエンジニア

  • ネットワークやサーバなどのWebの基礎知識を身につけることができる
  • なんとなく書いていた部分など、より正しい文書構造でマークアップすることができる

システムエンジニア

  • 動的な処理をサーバーサイドで担当するか、JavaScriptで実装するかなど、最適な実装方法を導くための選択肢が増える

デザイナー

  • レスポンシブWebデザインにおける実現方法をイメージしながらデザインできる
  • アニメーション等の実装イメージがつく

Webディレクター

  • 実現可能な技術を把握し、顧客に対してより具体的な提案ができるようになる
  • エンジニアに対して説得力ある提言ができるようになる

試験概要・対策

試験概要

  • 試験時間：90分（アンケート時間含む）
  • 問題数：約60問
  • 受験料：16,200円（税込）
  • 試験方式：CBT
  • 日時・会場：全国各地から自由に選択

合格ラインは約7割

出題範囲

  • Webの基礎知識
  • CSS
  • 要素
  • レスポンシブWebデザイン
  • APIの基礎知識

出題形式

  • 選択式
  • 記述式

出題例

HTML5の文書型宣言として間違っているものを2つ選びなさい。 
A． <!doctype>
B． <!doctype html>
C． < !DOCTYPE HTML>
D． <!DOCTYPE HTML >
E． <!DOCTYPE html PUBLIC "-//W3C//DTD HTML 4.01//EN">

正解：A、C

出題例

HTML5のtitle要素に関する記述のうち、間違っているものをすべて選びなさい。 
A． 1つのHTML文書に必ず1つ必要。
B． head要素内に配置する。
C． 複数は配置できない。
D． 要素内容にはテキストしか入れられない。
E． 条件によっては終了タグを省略できる。

正解：A、E

出題例

CSSのアニメーションの@keyframesの書式内において、「100%」の代わりに使用できるキーワードを記述しなさい。

正解：to

データベースに登録された英語住所が大文字小文字混在しているため、CSSを用い、すべて大文字で表示したい。text-transform プロパティで指定する値として適切なのは次のうちどれか。

A． uppercase
B． lowercase
C． capitalize
D． acronym
E． abbreviation

正解：A

p {
  text-transform: uppercase;
}

1つめのtr要素にのみスタイルを適用するセレクタをすべて選びなさい。

A． tr:first-child
B． tr:nth-child(1)
C． tr:nth-child(0n+1)
D． tr:nth-child(1n)
E． tr:nth-child(1n+0)

表の１行目だけに色をつける

<table>
<tr><td>1993年6月</td><td>HTML1.0</td></tr>
<tr><td>1995年11月</td><td>HTML 2.0</td></tr>
<tr><td>1997年12月</td><td>HTML 4.01</td></tr>
<tr><td>2014年10月</td><td>HTML 5</td></tr>
</table>

正解：A,B,C

tr:first-child {
  background: #eee;
}
tr:nth-child(1) {
  background: #eee;
}
tr:nth-child(0n+1) {
  background: #eee;
}
